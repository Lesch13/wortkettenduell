# Wortkettenduell

Ein browserbasiertes Wortspiel auf Deutsch.

## Spielprinzip

Bilde Wortketten, bei denen jedes neue Wort mit dem letzten Buchstaben des vorherigen beginnt — zum Beispiel: **Hund → Dachs → Schule → …**

## Starten

Einfach `index.html` im Browser öffnen — keine Installation nötig.
